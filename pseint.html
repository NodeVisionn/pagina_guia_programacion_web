¿Qué es PSeInt?

PseInt (Pseudocódigo Intérprete) es una herramienta de aprendizaje dirigida a la lógica de programación lanzada por primera vez el 29 de diciembre de 2003. Creada por Pablo Novara, su propósito principal es preparar y enseñarle a estudiantes los procesos de solución de algoritmos mendiante el pseudocódigo, y estructuras esenciales de los lenguajes de programación.

¿Por qué aprender PSeInt?

PSeInt permite desarrollar habilidades de programación de una manera simple y rápida, ofreciéndo herramientas visuales y lógicas para ejercerlas. Permite adquirir la experiencia para en un futuro adentrarse en otros lenguajes de programación.

¿Cómo obtener PSeInt?

Paso 1: Descargar el instalador. Se puede conseguir en la página oficial de PSeInt: pseint.sourceforge.net.

Paso 2: Ejecutar el instalador.

Paso 3: Seleccionar la ubicación de los archivos a descargar.

Estructuras y comandos

Cada algoritmo creado en PSeInt debe comenzar con "Algoritmo" y terminar con "FinAlgoritmo". Estos hacen parte de los comandos esenciales de un algoritmo, que se pueden reconocer, junto a las otros comandos, por su color azul y fuente en negrilla.

Escribir


Algoritmo
	Escribir "Hola Mundo!"
FinAlgoritmo

Leer


Algoritmo
	Escribir "Ingrese el valor de A."
	Leer a
	Escribir "Ingrese el valor de B."
	Leer b
	Escribir "Ingrese el valor de C."
	Leer c
	Escribir "A=",a," B=",b," C=",c;
FinAlgoritmo

Asignar


Algoritmo
	Definir cateto1,cateto2,hipotenuse Como Real
	Escribir "Ingrese el valor del primer cateto."
	Leer cateto1
	Escribir "Ingrese el valor del segundo cateto."
	Leer cateto2
	hipotenuse=[(cateto1)^2+(cateto2)^2]/2
	Escribir "La hipotenusa es igual a: ",hipotenuse;
FinAlgoritmo

Si-Entonces


Algoritmo
	Escribir "[-] Nota 1:"
	Leer nota1
	Escribir "[-] Nota 2:"
	Leer nota2
	Escribir "[-] Nota 3:"
	Leer nota3
	promedio=(nota1+nota2+nota3)/3
	Si promedio>=4 Entonces
		Escribir "[-] El estudiante aprobó, con un promedio de: ",promedio
	SiNo
		Escribir "[-] El estudiante reprobó, con un promedio de: ",promedio
	FinSi
FinAlgoritmo

Mientras


Algoritmo
	Definir i Como Entero
	i=1
	Mientras i<=10 Hacer
		Escribir i
		i<-i+1
	FinMientras
FinAlgoritmo

Repetir


Algoritmo
	Definir contador, veces como  entero 
	contador <- -10;
	Escribir  "escribe el numero de veces que quieres que se repita" 
	leer veces
	Repetir
		Escribir "Contador: ", contador;
		contador<- contador + 1; 
	Hasta Que  contador > veces
FinAlgoritmo

Para


Algoritmo sin_titulo
	Escribir "ingrese la cantidad:"; Leer n ;
	Dimension a[n];
	Para i<- 1 Hasta 10 Hacer
		Escribir "ingrese el dato", i; 
		Leer a[i];
	FinPara
FinAlgoritmo

Función


Funcion area <- CalcularAreaTriangulo(base, altura)
    area <- (base * altura) / 2
FinFuncion

Algoritmo principal
    area <- CalcularAreaTriangulo(5, 8)
    Escribir "El área del triángulo es: ", area
FinAlgoritmo

Diagramas de flujo
