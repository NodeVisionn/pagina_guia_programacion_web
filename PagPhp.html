Introducción a PHP

PHP (acrónimo recursivo de "PHP: Hypertext Preprocessor") es un lenguaje de programación del lado del servidor, especialmente diseñado para el desarrollo web. Fue creado en 1994 por Rasmus Lerdorf y se ha convertido en uno de los lenguajes más utilizados para crear sitios dinámicos e interactivos.

¿Por qué aprender PHP?

PHP es ampliamente usado en el desarrollo web debido a su facilidad de aprendizaje, amplia documentación y comunidad activa. Muchas plataformas populares como WordPress, Joomla y Drupal están desarrolladas con PHP. Además, es compatible con la mayoría de servidores y sistemas operativos.

Requisitos para comenzar

Antes de comenzar a programar en PHP, es recomendable contar con un servidor local que interprete el código PHP. Puedes instalar herramientas como XAMPP, WAMP o MAMP, que te permiten ejecutar tu código en tu propio computador sin necesidad de tener un servidor en línea. Para ver un tutorial presione la primera imagen que se encuentra al lado derecho.

Estructuras de control

PHP permite usar estructuras condicionales como if, else, y bucles como for, while y foreach. Estas estructuras permiten controlar el flujo del programa. Si Aún no le queda muy claro este tema, y quiere profundizar un poco más en estas estructuras, de click en la segunda imagen del lado derecho.

Conclusión

PHP es una excelente opción para quienes desean empezar a desarrollar sitios web dinámicos. Con práctica y dedicación, podrás crear aplicaciones web funcionales e incluso interactuar con bases de datos como MySQL. Esta guía es solo el comienzo, ¡te invitamos a seguir explorando!

Estructuras De Control

Primer script en PHP

Todo archivo PHP debe tener la extensión .php y su contenido debe estar dentro de etiquetas especiales. Aquí tienes un ejemplo básico:

<?php
  echo "¡Hola, mundo!";
?>

Este código imprimirá el mensaje "¡Hola, mundo!" en tu navegador. La función echo se usa para mostrar información en pantalla.

Variables en PHP

En PHP, las variables comienzan con el signo $ seguido del nombre de la variable. No es necesario declarar el tipo de dato, ya que PHP lo interpreta automáticamente. Ejemplo:

<?php
  $nombre = "Juan";
  $edad = 20;
  echo "Mi nombre es $nombre y tengo $edad años.";
?>

Estructura IF

<?php
$edad = 18;
if ($edad >= 18) {
  echo "Eres mayor de edad.";
}
?>

<?php
$numero = 5;
if ($numero > 0) {
  echo "El número es positivo.";
}
?>

Estructura IF...ELSE

<?php
$hora = 14;
if ($hora < 12) {
  echo "Buenos días";
} else {
  echo "Buenas tardes";
}
?>

<?php
$valor = 10;
if ($valor % 2 == 0) {
  echo "Es par";
} else {
  echo "Es impar";
}
?>

Estructura IF...ELSEIF...ELSE

<?php
$nota = 85;
if ($nota >= 90) {
  echo "Excelente";
} elseif ($nota >= 70) {
  echo "Aprobado";
} else {
  echo "Reprobado";
}
?>

<?php
$dia = "martes";
if ($dia == "lunes") {
  echo "Inicio de semana";
} elseif ($dia == "viernes") {
  echo "Fin de semana cerca";
} else {
  echo "Día normal";
}
?>

Estructura SWITCH

<?php
$opcion = 2;
switch ($opcion) {
  case 1:
    echo "Seleccionaste uno";
    break;
  case 2:
    echo "Seleccionaste dos";
    break;
  default:
    echo "Opción no válida";
}
?>

<?php
$color = "azul";
switch ($color) {
  case "rojo":
    echo "Color pasión";
    break;
  case "azul":
    echo "Color calma";
    break;
  default:
    echo "Color desconocido";
}
?>

Estructura WHILE

<?php
$i = 1;
while ($i <= 5) {
  echo "Número: $i <br>";
  $i++;
}
?>

<?php
$contador = 0;
while ($contador < 3) {
  echo "Iteración $contador <br>";
  $contador++;
}
?>

Estructura DO...WHILE

<?php
$j = 1;
do {
  echo "Valor: $j <br>";
  $j++;
} while ($j <= 3);
?>

<?php
$numero = 6;
do {
  echo "Número es $numero <br>";
  $numero--;
} while ($numero > 3);
?>

Estructura FOR

<?php
for ($i = 1; $i <= 5; $i++) {
  echo "Iteración: $i <br>";
}
?>

<?php
for ($x = 0; $x < 10; $x += 2) {
  echo "Número par: $x <br>";
}
?>

Estructura FOREACH

<?php
$nombres = ["Ana", "Luis", "Carlos"];
foreach ($nombres as $nombre) {
  echo "Hola $nombre <br>";
}
?>

<?php
$edades = ["Ana" => 20, "Luis" => 22];
foreach ($edades as $nombre => $edad) {
  echo "$nombre tiene $edad años <br>";
}
?>
