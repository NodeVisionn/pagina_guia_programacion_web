Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje fundamental de la web. Sirve para estructurar el contenido de las páginas, permitiendo definir textos, imágenes, enlaces, tablas y más.

¿Por qué aprender HTML?

Es el punto de partida para todo desarrollador web. Aprender HTML te permite entender cómo se construyen los sitios y cómo se combinan con CSS y JavaScript para lograr diseños dinámicos e interactivos.

Estructura básica de un documento HTML

Todo archivo HTML comienza con una estructura base que indica al navegador cómo debe interpretarlo:

<!doctype html>
<html>
<head>
  <title>Mi primera página</title>
</head>
<body>
  <h1>¡Hola, mundo!</h1>
  <p>Este es mi primer sitio web.</p>
</body>
</html>

Encabezados y párrafos

HTML tiene seis niveles de encabezado (<h1> hasta <h6>). Los párrafos se crean con la etiqueta <p>.

<h1>Título principal</h1>
<h2>Subtítulo</h2>
<p>Este es un párrafo con información sobre mi página.</p>

Listas

Las listas se pueden crear de dos tipos: ordenadas (<ol>) y no ordenadas (<ul>):

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<ol>
  <li>Abrir Visual Studio Code</li>

  <li>Crear un archivo .html</li>
  <li>Escribir tu código</li>
</ol>

Imágenes y enlaces

Las imágenes se insertan con la etiqueta <img>, y los enlaces con <a>:

<img src="foto.jpg" alt="Descripción de la imagen" width="300">
<a href="https://www.google.com">Visita Google</a>

Tablas básicas

HTML también permite crear tablas simples para organizar información:

<table border="1">
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
  </tr>
  <tr>
    <td>Thomas</td>
    <td>17</td>
  </tr>
</table>

Conclusión

HTML es la base de toda página web. Aprenderlo te permitirá crear la estructura de tus propios proyectos y, con el tiempo, podrás combinarlo con CSS para darle estilo y con JavaScript para hacerlo interactivo.
